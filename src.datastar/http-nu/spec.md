## What This Project Is

http-nu is a Nushell-scriptable HTTP server. You write route handlers as Nushell closures and it serves them over HTTP/1.1 or HTTPS. Integrates with cross.stream (xs) for persistence and Datastar for reactive frontends. Features hot reload, Brotli compression, static file serving, MiniJinja templates, syntax highlighting, and markdown rendering.
