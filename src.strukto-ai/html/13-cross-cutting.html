Shell Barrier

Source: typescript/packages/core/src/shell/barrier.ts

Restricts dangerous commands:

applyBarrier(shellNode, {
  blocked: ['rm -rf /', 'mkfs', 'dd', 'wget', 'curl'],
  allowed: ['cat', 'grep', 'sed', 'awk', 'cp', 'mv'],
})

Security Architecture

flowchart TD A[Agent command] --> B[Shell Parser] B --> C[Shell Barrier] C --> D{Blocked command?} D -->|Yes| E[Return error to agent] D -->|No| F[Execute normally] E --> G[Agent learns constraint] F --> H[Output to agent]

Secret Redaction Flow

flowchart LR A[Resource config] --> B[redactConfigWithSchema] B --> C[SecretStr fields masked] C --> D[Safe to log/store] D --> E{hasRedactedSecret?} E -->|Yes| F[Block output] E -->|No| G[Pass through]

Aha: The shell barrier prevents agents from executing destructive commands — rm -rf /, mkfs, dd — while allowing safe read operations.
