# pi-vs-claude-code

**Customized Pi instances showcasing advantages over Claude Code.**

pi-vs-claude-code (also called pi-vs-cc) is a collection of customized Pi coding agent instances that demonstrate what it looks like to hedge against the leader in the agentic coding market, Claude Code.

## Why

This extension showcases how you can customize the UI, agent orchestration tools, safety auditing, and cross-agent integrations in Pi to create a competitive alternative to Claude Code.

Watch the [video](https://youtu.be/f8cfH5XX-XU) to see Pi in action.

## Prerequisites

| Tool | Purpose | Install |
|------|---------|---------|
| **Bun** >= 1.3.2 | Runtime & package manager | [bun.sh](https://bun.sh) |
| **just** | Task runner | `brew install just` |

## What It Showcases

| Area | Customization |
|------|--------------|
| UI | Custom editor appearance and status bar |
| Agent orchestration | Multi-agent coordination patterns |
| Safety auditing | Security checks and permission controls |
| Cross-agent integrations | Integration with other agent ecosystems |
