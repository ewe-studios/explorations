pi-vs-claude-code

Customized Pi instances showcasing advantages over Claude Code.

pi-vs-claude-code (also called pi-vs-cc) is a collection of customized Pi coding agent instances that demonstrate what it looks like to hedge against the leader in the agentic coding market, Claude Code.

Why

This extension showcases how you can customize the UI, agent orchestration tools, safety auditing, and cross-agent integrations in Pi to create a competitive alternative to Claude Code.

Watch the video to see Pi in action.
