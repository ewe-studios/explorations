aipack Documentation

Jeremy Chone's Rust crate collection — 13 crates covering AI providers, JSON-RPC routing, SQL builders, MCP protocol support, and developer tools.

Getting Started

Start here: 00-overview.md — high-level introduction to all 13 crates and how they connect.

Documents

Foundation

  • 00-overview — Crate collection overview, key numbers, crate catalog
  • 01-architecture — Module structures, dependency diagrams for all crates

AI Layer

  • 02-genai — Multi-provider AI client (19 providers: OpenAI, Anthropic, Gemini, Ollama, etc.) — chat API, embeddings, tool use, streaming, adapter system
  • 06-agentic — MCP protocol client/server (HTTP + stdio transports, tools, resources, prompts, sampling, capability negotiation)

Data Layer

  • 04-sqlb — Simple, expressive SQL builder (SELECT/INSERT/UPDATE/DELETE) with sqlx integration
  • 05-modql — Model Query Language (filter operators, JSON filter syntax, SeaQuery/SQLite adapters, pagination)

Transport Layer

  • 03-rpc-router — JSON-RPC 2.0 router with procedural macros (RpcResource, RpcParams, RpcHandlerError)

Dev Tools

  • 07-udiffx — LLM-optimized unified diff parser/applier with patch completion
  • 08-utilities — Utility crates: simple-fs (filesystem API), uuid-extra (UUID encoding), pretty-sqlite, dinf, htmd (HTML→Markdown), webdev, webtk
