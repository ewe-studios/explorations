# TypeGPU Rust Revision Plan

## Overview

This document outlines what it would take to replicate TypeGPU's functionality in Rust. The goal is to create a type-safe Rust crate that provides similar abstractions over wgpu (the WebGPU implementation for Rust).

## Core abstractions

### 1. Type System Equivalents

**TypeScript Pattern**: `$repr` symbol for type inference

```typescript
// TypeScript
interface F32 {
  type: 'f32';
  [$repr]: number;
}
type Infer<T> = T[typeof $repr];
```

**Rust Equivalent**: Associated types and marker traits

```rust
// Rust
pub trait ShaderType {
    type CpuRepr: Sized + Clone;
    type GpuRepr: Sized + Clone;
    type PartialRepr: Sized + Clone = Self::CpuRepr;

    const SIZE: usize;
    const ALIGNMENT: usize;

    fn write_cpu(writer: &mut BufferWriter, value: &Self::CpuRepr);
    fn read_cpu(reader: &mut BufferReader) -> Self::CpuRepr;
}

// Implementation for f32
pub struct F32;

impl ShaderType for F32 {
    type CpuRepr = f32;
    type GpuRepr = f32;
    type PartialRepr = f32;

    const SIZE: usize = 4;
    const ALIGNMENT: usize = 4;

    fn write_cpu(writer: &mut BufferWriter, value: &f32) {
        writer.write_f32(*value);
    }

    fn read_cpu(reader: &mut BufferReader) -> f32 {
        reader.read_f32()
    }
}
```

### 3. Struct Schema System

**TypeScript Pattern**: Runtime schemas with compile-time inference

```typescript
// TypeScript
const Particle = struct({
  position: vec3f,
  velocity: vec3f,
  mass: f32,
});
type ParticleType = Infer<typeof Particle>;
```

**Rust Equivalent**: Derive macros and trait implementations

```rust
// Rust
use typegpu::ShaderType;
use typegpu_macros::ShaderType;

#[derive(ShaderType, Clone, Copy)]
#[repr(C)]
pub struct Particle {
    pub position: Vec3F,
    pub velocity: Vec3F,
    pub mass: f32,
}

// Generated by derive macro:
impl ShaderType for Particle {
    type CpuRepr = Self;
    type GpuRepr = Self;

    const SIZE: usize = 32;  // 12 + 12 + 4 + 4 (padding)
    const ALIGNMENT: usize = 16;

    fn write_cpu(writer: &mut BufferWriter, value: &Self) {
        // Alignment-aware serialization
        writer.write_aligned(value.position);
        writer.write_aligned(value.velocity);
        writer.write_f32(value.mass);
        writer.write_padding(4);  // Explicit padding
    }

    fn read_cpu(reader: &mut BufferReader) -> Self {
        Self {
            position: reader.read_aligned(),
            velocity: reader.read_aligned(),
            mass: reader.read_f32(),
        }
    }
}
```

### 4. Vector Types with Swizzle

**TypeScript Pattern**: Swizzle getters on vector classes

```typescript
// TypeScript
const v = vec3f(1, 2, 3);
const swizzled = v.xy;  // { x: 1, y: 2 }
```

**Rust Equivalent**: Extension traits and const generics

```rust
// Rust
#[derive(Clone, Copy, Debug)]
pub struct Vec3F {
    pub x: f32,
    pub y: f32,
    pub z: f32,
}

// Swizzle traits
pub trait Swizzle2<T> {
    type Output;
    fn xx(&self) -> Self::Output;
    fn xy(&self) -> Self::Output;
    fn xz(&self) -> Self::Output;
    fn yx(&self) -> Self::Output;
    fn yy(&self) -> Self::Output;
    fn yz(&self) -> Self::Output;
    fn zx(&self) -> Self::Output;
    fn zy(&self) -> Self::Output;
    fn zz(&self) -> Self::Output;
}

pub trait Swizzle3<T> {
    type Output;
    fn xyz(&self) -> Self::Output;
    fn xzy(&self) -> Self::Output;
    // ... more combinations
}

// Implementation
impl Swizzle2<Vec3F> for Vec3F {
    type Output = Vec2F;

    fn xx(&self) -> Vec2F { Vec2F::new(self.x, self.x) }
    fn xy(&self) -> Vec2F { Vec2F::new(self.x, self.y) }
    fn xz(&self) -> Vec2F { Vec2F::new(self.x, self.z) }
    // ... more swizzles
}

// Usage
let v = Vec3F::new(1.0, 2.0, 3.0);
let swizzled = v.xy();  // Vec2F::new(1.0, 2.0)
```

### 5. WGSL Type Literals

**TypeScript Pattern**: WGSL types as TypeScript types

```typescript
// TypeScript
fn add(a: f32, b: f32): f32 {
  return a + b;
}
```

**Rust Equivalent**: Newtype wrappers and trait bounds

```rust
// Rust
// WGSL type wrappers
pub struct F32(f32);
pub struct I32(i32);
pub struct U32(u32);
pub struct Vec3F(Vec3<f32>);

// Shader function trait
pub trait ShaderFn<Args, Return> {
    const WGSL_SIGNATURE: &'static str;

    fn to_wgsl(&self) -> String;
}

// Implementation
impl ShaderFn<(F32, F32), F32> for fn(F32, F32) -> F32 {
    const WGSL_SIGNATURE: &'static str = "fn(f32, f32) -> f32";

    fn to_wgsl(&self) -> String {
        // Generate WGSL function
        format!("fn add_{}(a: f32, b: f32) -> f32 {{ return a + b; }}", unique_id())
    }
}
```

## wgpu Integration

### Device Initialization

```rust
// Rust equivalent of tgpu.init()
pub struct DeviceConfig {
    pub label: Option<String>,
    pub features: wgpu::Features,
    pub limits: wgpu::Limits,
}

pub async fn init(config: Option<DeviceConfig>) -> Result<Context, InitError> {
    let instance = wgpu::Instance::new(wgpu::InstanceDescriptor {
        backends: wgpu::Backends::all(),
        ..Default::default()
    });

    let adapter = instance.request_adapter(&wgpu::RequestAdapterOptions {
        power_preference: wgpu::PowerPreference::default(),
        force_fallback_adapter: false,
        compatible_surface: None,
    }).await.ok_or(InitError::NoAdapter)?;

    let (device, queue) = adapter.request_device(
        &wgpu::DeviceDescriptor {
            label: config.as_ref().and_then(|c| c.label.as_deref()),
            features: config.as_ref().map(|c| c.features).unwrap_or_default(),
            limits: config.as_ref().map(|c| c.limits).unwrap_or_default(),
        },
        None,
    ).await?;

    Ok(Context::new(device, queue))
}

pub struct Context {
    device: wgpu::Device,
    queue: wgpu::Queue,
    command_encoder: Option<wgpu::CommandEncoder>,
}
```

### Buffer Creation

```rust
// Rust equivalent of tgpu.buffer()
pub struct BufferBuilder<T: ShaderType> {
    _marker: PhantomData<T>,
    initial_data: Option<T::CpuRepr>,
    label: Option<String>,
    usage: wgpu::BufferUsages,
}

impl<T: ShaderType> BufferBuilder<T> {
    pub fn new() -> Self {
        Self {
            _marker: PhantomData,
            initial_data: None,
            label: None,
            usage: wgpu::BufferUsages::COPY_DST | wgpu::BufferUsages::COPY_SRC,
        }
    }

    pub fn initial_data(mut self, data: T::CpuRepr) -> Self {
        self.initial_data = Some(data);
        self
    }

    pub fn usage(mut self, usage: wgpu::BufferUsages) -> Self {
        self.usage = usage;
        self
    }

    pub fn build(self, ctx: &Context) -> Result<Buffer<T, Uniform>, BufferError> {
        let size = self.initial_data
            .as_ref()
            .map(|_| T::SIZE as u64)
            .unwrap_or(T::SIZE as u64);

        let buffer = ctx.device.create_buffer(&wgpu::BufferDescriptor {
            label: self.label.as_deref(),
            size,
            usage: self.usage,
            mapped_at_creation: self.initial_data.is_some(),
        });

        if let Some(data) = self.initial_data {
            let mut writer = BufferWriter::new(buffer.slice(..));
            T::write_cpu(&mut writer, &data);
        }

        Ok(Buffer {
            _marker: PhantomData,
            inner: buffer,
            device: ctx.device.clone(),
        })
    }
}

// Convenience function
pub fn buffer<T: ShaderType>() -> BufferBuilder<T> {
    BufferBuilder::new()
}
```

### Buffer Operations

```rust
// Write operation
impl<T: ShaderType, U> Buffer<T, U> {
    pub fn write(&mut self, data: &T::CpuRepr) {
        let mut writer = BufferWriter::new(self.inner.slice(..));
        T::write_cpu(&mut writer, data);
        // Note: In real implementation, would need staging buffer
    }
}

// Read operation (async)
impl<T: ShaderType, U> Buffer<T, U> {
    pub async fn read(&self) -> Result<T::CpuRepr, ReadError> {
        // Create staging buffer
        let staging_buffer = self.device.create_buffer(&wgpu::BufferDescriptor {
            label: Some("staging buffer"),
            size: T::SIZE as u64,
            usage: wgpu::BufferUsages::COPY_DST | wgpu::BufferUsages::MAP_READ,
            mapped_at_creation: false,
        });

        // Copy to staging buffer
        let mut encoder = self.device.create_command_encoder(
            &wgpu::CommandEncoderDescriptor {
                label: Some("read encoder")
            }
        );

        encoder.copy_buffer_to_buffer(
            &self.inner, 0,
            &staging_buffer, 0,
            T::SIZE as u64,
        );

        let command_buffer = encoder.finish();
        self.device.queue().submit(Some(command_buffer));

        // Map and read
        staging_buffer.slice(..).map_async(wgpu::MapMode::Read, |_| {}).await?;

        let reader = BufferReader::new(staging_buffer.slice(..).get_mapped_range());
        let data = T::read_cpu(&reader);

        staging_buffer.unmap();
        staging_buffer.destroy();

        Ok(data)
    }
}
```

```rust
// For dynamic shader generation
pub struct ShaderBuilder {
    functions: Vec<String>,
    structs: Vec<String>,
    bindings: Vec<Binding>,
}

impl ShaderBuilder {
    pub fn new() -> Self {
        Self {
            functions: Vec::new(),
            structs: Vec::new(),
            bindings: Vec::new(),
        }
    }

    pub fn add_function<F: ShaderFn>(&mut self, func: F) -> &mut Self {
        self.functions.push(func.to_wgsl());
        self
    }

    pub fn build(self) -> ShaderModule {
        let wgsl = format!(
            "{}\n\n{}",
            self.structs.join("\n"),
            self.functions.join("\n")
        );

        ShaderModule { wgsl }
    }
}
```

## Ownership Considerations

### Resource Ownership

```rust
// TypeGPU (TypeScript) - GC handles cleanup

// Rust - Explicit ownership and Drop

    .initial_data(particle)
    .build(&ctx)?;

// Explicit cleanup

// Or automatic via Drop
impl<T: ShaderType, U> Drop for Buffer<T, U> {
    fn drop(&mut self) {
        self.inner.destroy();
    }
```

## Safe vs Unsafe Abstractions

### Safe Abstractions

```rust

pub struct TypedBuffer<T: ShaderType> {
    inner: wgpu::Buffer,
    _marker: PhantomData<T>,

impl<T: ShaderType> TypedBuffer<T> {
    // Safe - type ensures correct serialization

        let mut writer = BufferWriter::new(self.inner.slice(..));
        T::write_cpu(&mut writer, data);
    }

    // Safe - type ensures correct deserialization

        // ... staging buffer pattern
    }

```

### Unsafe Internals

```rust
// Unsafe used internally for performance
// Compiled I/O using pre-generated code
pub unsafe fn compiled_write<T: ShaderType>(
    ptr: *mut u8,
    data: &T::CpuRepr,
) {
    // Pre-computed offsets for fast serialization
    // Only safe if called with correct types
    *(ptr as *mut T::CpuRepr) = *data;
}

// Memory-mapped buffer access
pub struct MappedBuffer<'a, T: ShaderType> {
    ptr: *mut T::CpuRepr,
    _marker: PhantomData<&'a mut T>,
}

impl<'a, T: ShaderType> MappedBuffer<'a, T> {
    // Unsafe - requires proper synchronization
    pub unsafe fn get(&self) -> &T::CpuRepr {
        &*self.ptr
    }

    pub unsafe fn get_mut(&mut self) -> &mut T::CpuRepr {
        &mut *self.ptr
    }
}
```

## Key Differences from TypeScript

| Feature | TypeScript TypeGPU | Rust Equivalent |
|---------|-------------------|-----------------|
| Type inference | `$repr` symbol | Associated types |
| Runtime schemas | Object with `[$repr]` | Derive macros |
| Phantom types | Type parameters | PhantomData<T> |
| GC | Automatic | Drop trait |
| Template literals | `` tgpu.fn`...` `` | Proc macros |
| eval-based I/O | `new Function()` | Unsafe ptr operations |
| WeakMap memoization | `WeakMap` | `Weak<HashMap>` |
| Swizzle | Getters on class | Extension traits |

## Challenges and Solutions

### Challenge 1: Runtime Type Information
**Problem**: Rust erases types at runtime, but TypeGPU needs runtime type info.
**Solution**: Use trait objects and TypeId for dynamic dispatch when needed.

### Challenge 2: Shader Generation
**Problem**: Rust macros run at compile time, but TypeGPU supports runtime shader generation.
**Solution**: Provide both compile-time macros and runtime builder pattern.

### Challenge 3: Memory Safety
**Problem**: GPU memory access patterns are inherently unsafe.
**Solution**: Safe abstractions with unsafe internals, rigorous testing.

### Challenge 4: Borrow Checker
**Problem**: GPU resources have complex lifetimes.
**Solution**: Careful use of lifetimes, Rc/RefCell for shared ownership.

## Example Usage

```rust
use typegpu::*;
use typegpu::wgsl::*;

// Define shader types
#[derive(ShaderType, Clone, Copy)]
#[repr(C)]
struct Config {
    time: f32,
    resolution: Vec2F,
}

#[derive(ShaderType, Clone, Copy)]
#[repr(C)]
struct Particle {
    position: Vec3F,
    velocity: Vec3F,
    mass: f32,
}

// Define compute shader
#[derive(ShaderFunction)]
#[workgroup_size(64)]
fn update_particles(
    #[storage(read_write)] particles: &[Particle],
    #[uniform] config: &Config,
    #[builtin(global_invocation_id)] id: UVec3,
) {
    let idx = id.x as usize;
    if idx < particles.len() {
        particles[idx].position += particles[idx].velocity * config.time;
    }
}

// Main application
#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Initialize
    let ctx = typegpu::init(None).await?;

    // Create buffers
    let config_buffer = buffer::<Config>()
        .initial_data(Config {
            time: 0.016,
            resolution: Vec2F::new(800.0, 600.0),
        })
        .usage(wgpu::BufferUsages::UNIFORM | wgpu::BufferUsages::COPY_DST)
        .build(&ctx)?;

    let particles: Vec<Particle> = (0..1000)
        .map(|i| Particle {
            position: Vec3F::new(i as f32, 0.0, 0.0),
            velocity: Vec3F::new(0.0, 1.0, 0.0),
            mass: 1.0,
        })
        .collect();

    let particle_buffer = buffer::<Particle>()
        .initial_data(particles)
        .usage(wgpu::BufferUsages::STORAGE | wgpu::BufferUsages::COPY_DST)
        .build(&ctx)?;

    // Create pipeline
    let pipeline = ctx.create_compute_pipeline(&update_particles::SHADER)?;

    // Dispatch
    let mut encoder = ctx.create_command_encoder();
    let mut pass = encoder.begin_compute_pass();
    pass.set_pipeline(&pipeline);
    pass.set_bind_group(0, &config_buffer.bind_group());
    pass.set_bind_group(1, &particle_buffer.bind_group());
    pass.dispatch_workgroups((1000 + 63) / 64, 1, 1);
    pass.end();

    ctx.submit(encoder);

    // Read results
    let updated_particles = particle_buffer.read().await?;

    Ok(())
}
```
