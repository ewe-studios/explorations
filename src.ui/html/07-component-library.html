OpenUI -- Component Library

The react-ui package provides 60+ prebuilt React components organized into categories: layout, data display, forms, actions, content, and system. Two built-in libraries are exported: openuiLibrary (standalone) and openuiChatLibrary (chat-focused with component groups).

Aha: Components are registered in the library by their exported name, and the materializer uses JSON Schema for each component's props. The schema serves three purposes: (1) validation — unknown props are dropped with errors, (2) default values — missing props get schema defaults, (3) positional mapping — the LLM can write
