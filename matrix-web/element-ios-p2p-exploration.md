# Sub-Project Exploration: Element iOS P2P

## Overview

Element iOS P2P is an experimental fork of Element iOS that embeds Dendrite with Pinecone peer-to-peer networking, enabling serverless Matrix communication on iOS. Similar to Element Android P2P, it demonstrates Matrix's P2P capabilities.

## Key Insights

- Experimental fork of element-ios with embedded Dendrite
- Pinecone overlay network for P2P federation
- Demo/research purposes only
- CocoaPods dependency management (Podfile present)
- Same architecture as element-ios but with embedded homeserver
