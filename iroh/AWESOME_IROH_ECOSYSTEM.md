# Awesome Iroh: Ecosystem Overview

## Overview

`awesome-iroh` is a curated list of projects and resources building with iroh, the peer-to-peer networking stack from number 0. It serves as a directory of the growing iroh ecosystem.

**Repository:** https://github.com/n0-computer/awesome-iroh
**Badge:** [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

## Contributing

### How to Add Your Project

1. Fork the awesome-iroh repository
2. Add your project to the appropriate category in `readme.md`
3. Follow the existing format:
   ```markdown
   #### Project Name
   - **Repository:** URL
   - **Description:** Brief description
   - **Use of iroh:** How it uses iroh
   ```
4. Submit a pull request

### Guidelines

- Project must use iroh meaningfully
- Include working repository or website
- Provide clear description
- Specify how iroh is used

## Ecosystem Trends

### Growing Areas

1. **Local-first software**: Applications that work offline-first
2. **AI infrastructure**: Decentralized compute for AI
3. **Developer tools**: P2P alternatives to cloud services
4. **Gaming**: Multiplayer games without servers

### Emerging Patterns

1. **No-cloud architecture**: Full P2P without central servers
2. **Hybrid sync**: Combination of P2P and optional cloud backup
3. **Edge compute**: Distributed compute at network edge
4. **Sovereign software**: User-controlled infrastructure
